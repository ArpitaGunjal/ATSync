# ATSync
ATSync is a web-based resume screening and candidate filtering system designed to help companies streamline their recruitment process. In this system, the admin (company/HR) can post job opportunities and define the required skills for each job. Candidates can register, view available job openings, and apply by uploading their resumes. The system scans the uploaded resumes and compares them with the job requirements to generate an ATS (Applicant Tracking System) score. The admin can view all applicants, analyze their scores, and filter the most suitable candidates based on the requirements. After the selection process, the system notifies candidates with their results, ATS score, and suggestions for improvement. This platform helps automate resume screening and makes the recruitment process faster and more efficient.
